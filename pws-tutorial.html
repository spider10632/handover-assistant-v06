1. 登入方式

  1. 打開系統首頁，先輸入「伺服器名稱」。
  2. `test` 伺服器需輸入密碼登入（帳號對應角色）。
  3. 登入成功後，右上會顯示操作介面，可開始新增/查詢/派單。
建議測試流程：先用 testmanager 指派任務，再用 teststaff 驗證接單顯示與通知。

2. 權限分級（已實作）

角色 可做 限制
ADMIN 全部功能 無
MANAGER 全部功能（含指派） 無
STAFF 新增、修改、完成、查詢、看任務 不可指派、不可一鍵複製、不可置頂、不可刪除、不可匯出報表

3. 新增待辦與派單

  1. 在「新增交接待辦」選主分類與子分類。
  2. 填寫事項名稱、時間（可跨日）、填寫人、交接說明。
  3. 在「指派給」選擇處理人員（不會出現 admin 帳號）。
  4. 按「新增待辦」儲存。
若「指派給」只有「未指定（公用）」：通常是登入過期或快取舊版，請重新登入並重整頁面。

4. 當日事項與處理流程

  • 「置頂區」顯示 pinned 任務；一般任務顯示在當日事項清單。
  • 點「完成」會要求輸入完成人。
  • 已完成任務會自動取消置頂並隱藏於當日待辦區（依目前規則）。
  • 點「修改」會跳到新增區塊並帶入原內容。

5. 日期查詢與報表匯出

  1. 在「日期查詢」可用日期、狀態、主分類、關鍵字篩選。
  2. 查詢後會展開查詢結果。
  3. 可匯出 Word/Excel（STAFF 角色會停用）。
  4. 未選日期匯出時，會依「未來待辦事項」規則輸出。

6. 手機版與通知（PWA）

  • 手機開啟網站後，可加到主畫面（PWA 安裝）。
  • 在「下一件待辦」板塊可開啟通知授權。
  • 派單到期時，若裝置已訂閱推播，可收到網頁通知。
iPhone Safari 分享 → 加入主畫面 → 從主畫面啟動
Android Chrome 選單 → 安裝應用程式
通知前提 需先授權通知且完成雲端訂閱

7. 常見問題（FAQ）

  • Q：MANAGER 不能指派？
    A：請重新登入後再試，並確認伺服器為 `test`、帳號為 manager/admin。
  • Q：為什麼看不到 testorder？
    A：admin 已從指派清單隱藏，不可被派單。
  • Q：STAFF 為何按鈕灰色？
    A：屬於權限限制，按鈕保留但停用為正常行為。
  • Q：手機版版面亂掉？
    A：請關閉分頁重開，或清除瀏覽器快取後重載最新版。
