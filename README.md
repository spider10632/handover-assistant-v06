# 工作交接助手（Web 版）

一個可直接在瀏覽器使用的交接工具，提供：
- 新增/管理交接待辦
- 依日期查詢待辦
- 依時間提醒下一件待辦
- 一鍵匯出 Word 檔（`.docx`，失敗時自動改 `.doc` 相容模式）

## 使用方式

1. 進入資料夾後，直接開啟 `index.html`。
2. 若要使用瀏覽器通知，點選「授權通知」並允許權限。
3. 建議以 `localhost` 開啟（通知相容性較佳）。
4. 新增待辦時需先選擇主分類（廣場、包裹代收、車輛安排、餐飲部、待回覆信件、公告）。
5. 子分類目前在主分類為「廣場」、「包裹代收」或「車輛安排」時啟用：
   - 廣場：保留車位、其他
   - 包裹代收：團體、散客
   - 車輛安排：禮賓車、計程車
6. 分類尚未選完整前，其他輸入欄位與「新增待辦」按鈕會鎖定。
7. 「截止時間」旁可勾選「置頂顯示」，置頂中的待辦會在「待辦清單」自動排到最上方。
8. 截止時間可手動輸入，也可留空不填。
   - 可切換「全日」按鈕（只選日期，顯示為全天）
9. 「當日事項」板塊會顯示今天的事項清單與完成統計。
   - 預設為「全部主分類」
   - 依截止時間排序（由早到晚）
   - 最上方為置頂區（顯示今日置頂事項）
10. 右上角浮動「下一件待辦提醒」會集中顯示 30 分鐘內與 1 小時內的待辦。
11. 事件標記完成時，系統會要求填寫「完成人」。
12. 新增區塊提供「清空」按鈕，可一鍵清空目前輸入內容。

## 檔案結構

- `index.html`：頁面結構
- `styles.css`：版面與樣式
- `app.js`：待辦邏輯、日期查詢、提醒、Word 匯出

## 注意事項

- 待辦資料儲存在瀏覽器 `localStorage`。
- 清除瀏覽器網站資料會一起刪除待辦資料。
